Buttons

Use buttons throughout applications using the btn-peach-*colour*-*shade*class.

Do's and Don'ts

  • Do use disabled attribute if button cannot be clicked.
  • Do use same colour button as chosen navbar.
  • Do not alter button sizes.

Filled Example Usage









<button class="btn-peach-blue-med"> Submit </button>
<button class="btn-peach-green-dark"> Submit </button>

Outlined Example Usage

<button class="btn-peach-orange-dark-outline"> Submit </button>
<button class="btn-peach-sand-dark-outline"> Submit </button>

Navbar

Use the same navbar throughout applications using the nav-peach-*colour* class.

Do's and Don'ts

  • Do make sure to use the collapsable navbar for mobile.
  • Do not change the height of navbar
  • Do not change the brand logo padding. Must adhere to clear space rule.
